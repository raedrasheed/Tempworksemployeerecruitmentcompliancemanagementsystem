# Phase 1 Preflight Reports

The files in this directory are generated by `npm run saas:phase1-preflight` against the database referenced by `DATABASE_URL`.

**The committed files are the output of running preflight against the fixture in `backend/scripts/saas/phase1/__fixture__/seed.sql`**, not against production. They demonstrate the end-to-end shape of the audit; they are NOT real prod findings.

To produce real findings:

```sh
DATABASE_URL=postgres://prod-replica/... \
  npm --prefix backend run saas:phase1-preflight
```

Reports are overwritten on each run. Treat the JSON files as the machine-readable source; the Markdown files as the human-readable summary.
